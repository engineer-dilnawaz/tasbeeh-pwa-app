# Tasbeeh PWA — Divine Source Architecture (Empty Slate)

This branch has been purged of all legacy source code to provide a 100% clean starting point for **Divine Design System 2.0**.

## 🏗️ Folder Structure

- `src/shared/design-system/`: Centralized Tokens and UI Primitives.
- `src/shared/components/`: Common UI components shared across features.
- `src/app/`: Application shell, providers, and routing.
- `src/features/`: Domain-specific business logic.
- `src/pages/`: Route-level entry points.
- `src/services/`: External integrations (Firebase, API).
- `src/shared/utils/`: Global utility functions.
- `src/shared/types/`: Global TypeScript interfaces.
- `src/shared/styles/`: Global CSS and Tailwind configuration.

## 🛠️ Constraints
- **Styling**: Tailwind CSS & DaisyUI ONLY.
- **Components**: Framer Motion for animations.
- **Corners**: `corner-smoothing` for premium Squircles.
